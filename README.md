# Corneish Zen ZMK Keymap

36-key layout ported from QMK Vial with home row mods and combos.

## Combos

| Keys    | Output    |
|---------|-----------|
| H + J   | `<`       |
| K + L   | `>`       |
| J + K   | `:`       |
| V + B   | ESC       |
| F + G   | TAB       |
| D + F   | BACKSPACE |
| N + M   | ENTER     |

## Bluetooth

On Hyper layer (bottom row left side):
- **Z** = Clear current profile bond
- **X** = Profile 0
- **C** = Profile 1
- **V** = Profile 2
- **B** = Profile 3

## Flashing

1. Put keyboard half in bootloader mode (double-tap reset)
2. Copy `.uf2` file to the mounted `CORNEISHZEN` drive
3. Flash left side first, then right side
